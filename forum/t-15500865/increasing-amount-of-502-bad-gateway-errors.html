Increasing amount of 502 (Bad Gateway) Errors
BlazingGold 31 Dec 2022 14:58

Hi!

I've been using this website for a while now, and as of recent I've noticed a lot of 502 errors. I saw one mentioned already with Fire Genasi, but it's covered a lot of different pages - spell:summon-aberration, spells:wizard, etc. Many spells and other options seem to frequently return these errors, some lasting for several days at a time.

From what I know about these, it's possible that they can be from my end instead of the server's - I have been able to access some of those pages while in Incognito mode. But, since I'm not the only one, I was curious about what others have found and think is happening.
